Lenguajes de Marcas

HTML CSS XML XPath

HTML

HTML(HyperText Markup Languaje)es un lenguaje de marcas que se utilizan para estructurar contenido en la web.
Permite definir la estructura y el significado de una página web meduiante etiquetas.

CSS

CSS(Cascading Style Sheets)es un lenguaje de marcas utilizado para dar estilo y presentacion a documentos HTML.
Permite controlar la apariencia de los elementos de una página web.

XML

XML(eXtensible Markup Language) es un lenguaje de marcas que se utiliza para almacenar y transportar datos.
Es extensible y permite definir etiquetas personalizadas para estructurar información de manera jerárquica.

XPath

XPath es un lenguaje de marcas utilizado para navegar y consultar documentos XML.
Permite seleccionador nodos especificos dentro de una estructura XML utilizando rutas de acceso.
